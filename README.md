# ProfessionalPractice2
A chatbot designed to help users diagnose any problems that are currently being had with their computer system.

to run the chatbot:
download the files into a folder labelled 'chatbot' on your 'C:' drive
open command prompt and type the command 

'python C:/chatbot/Main.py'

When greeted with a login prompt, the base username and password is:

'Username'

'Password'

These can later be updated and deleted once onto the software

To stop the sofwtare running simply press 'CTRL + C'
