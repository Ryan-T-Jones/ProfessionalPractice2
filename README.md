# ProfessionalPractice2
A chatbot designed to help users diagnose any problems that are currently being had with their computer system.

to run the chatbot:\n
download the files into a folder labelled 'chatbot' on your 'C:' drive\n
open command prompt and type the command 'python C:/chatbot/Main.py'\n

When greeted with a login prompt, the base username and password is:\n
'Username'\n
'Password'\n
These can later be updated and deleted once onto the software
